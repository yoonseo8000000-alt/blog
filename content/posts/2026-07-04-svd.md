## 1. 특이값 분해 예제

### Positive Semidefinite Matrix (PSDM)

대칭행렬 $A$ ($A^T = A$)가 모든 벡터 $x$에 대하여 $x^TAx \geq 0$이면 $A$를 **양의 준정부호 행렬(PSDM)**이라 한다.

**임의의 행렬 $A$에 대해 $A^TA$, $AA^T$는 항상 PSDM이다:**

$$x^TA^TAx = (Ax)^T(Ax) \geq 0, \qquad x^TAA^Tx = (A^Tx)^T(A^Tx) \geq 0$$

$n \times n$ 행렬 $A$에 대해 다음 네 명제는 동치다.
1. $A$가 양의 준정부호 행렬이다
2. $A$의 모든 eigenvalue는 음의 실수가 아니다
3. 행렬 $U$가 존재해서 $A = U^TU$이다
4. $A$의 모든 sub-determinant는 음의 실수가 아니다

또한 대칭행렬의 서로 다른 고윳값에 대응하는 고유벡터는 서로 직교하고 ($v_1 \cdot v_2 = 0$), 정규직교행렬 $Q$로 $A = Q^TDQ$ 형태로 대각화할 수 있다.

### 예제 1: $A = \begin{pmatrix} 1 & 1 & 0 \\ 0 & 0 & 1 \end{pmatrix}$

$A^TA$의 고윳값을 구하고($\lambda = 2, 1, 0$), 각 고유벡터 $v_i$에 대해 $u_i = \frac{1}{\sqrt{\lambda_i}}Av_i$로 $U$를 구성하면:

$$U^TAV = \begin{pmatrix}\sqrt{2}&0&0\\0&1&0\end{pmatrix}$$

```python
A1 = np.array([[1, 1, 0], [0, 0, 1]], dtype=float)

# A^T A 계산 (대칭, 고유값 = σ²)
AtA = A1.T @ A1
eigvals, eigvecs = np.linalg.eigh(AtA)  # eigh: 대칭행렬 전용
idx = np.argsort(eigvals)[::-1]         # 내림차순 정렬
eigvals, eigvecs = eigvals[idx], eigvecs[:, idx]

# numpy SVD로 검증
U, S, Vt = np.linalg.svd(A1, full_matrices=True)

# 복원 검증: U Λ V^T = A
Lambda = np.zeros((U.shape[0], Vt.shape[0]))
for i, s in enumerate(S):
    Lambda[i, i] = s
A_reconstructed = U @ Lambda @ Vt
print('원래 A와 일치:', np.allclose(A1, A_reconstructed))  # True
```

### 예제 2: $B = \begin{pmatrix}1&1\\1&0\\0&1\end{pmatrix}$

$B^TB$의 고윳값은 $\lambda = 3, 1$이고, $u_1, u_2$는 $u_i = \frac{1}{\sqrt{\lambda_i}}Bv_i$로, $u_3$는 $u_1 \times u_2$(벡터곱)로 구한다:

$$U^TBV = \begin{pmatrix}\sqrt{3}&0\\0&1\\0&0\end{pmatrix}$$

```python
B = np.array([[1, 1], [1, 0], [0, 1]], dtype=float)
U2, S2, Vt2 = np.linalg.svd(B, full_matrices=True)

# Σ: m×n 대각 (특이값)
Sigma2 = np.zeros((3, 2))
for i, s in enumerate(S2):
    Sigma2[i, i] = s

result = U2.T @ B @ Vt2.T  # 대각행렬 Σ와 일치해야 함
```

### 예제 3: $A = \begin{pmatrix}4&0\\3&-5\end{pmatrix}$

$A^TA = \begin{pmatrix}25&-15\\-15&25\end{pmatrix}$의 특성방정식을 풀면 $\lambda = 40, 10$, 특이값은 $\sigma_1=\sqrt{40}=2\sqrt{10}$, $\sigma_2=\sqrt{10}$이다.

```python
A3 = np.array([[4, 0], [3, -5]], dtype=float)
U3, S3, Vt3 = np.linalg.svd(A3)

result3 = U3.T @ A3 @ Vt3.T
print('일치 여부:', np.allclose(result3, np.diag(S3)))  # True
```

## 2. 특이값 분해 이론

**SVD 정의**: $m \times n$ 행렬 $A$는 다음과 같이 분해된다.

$$A = U\Lambda V^T$$

| 기호 | 크기 | 성질 |
|---|---|---|
| $U$ | $m \times m$ | 직교행렬: $UU^T = U^TU = I$ |
| $\Lambda$ | $m \times n$ | 대각행렬 |
| $V$ | $n \times n$ | 직교행렬: $VV^T = V^TV = I$ |

$p = \min(m,n)$일 때 $\Sigma = \text{diag}(\sigma_1, \ldots, \sigma_p)$이고 $\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_p \geq 0$을 **특이값**이라 한다.

**핵심 관계**: $A = U\Lambda V^T$이면

$$A^TA = V\Lambda^TU^TU\Lambda V^T = V\Lambda^T\Lambda V^T$$

즉 $\Lambda_1 = \Lambda^T\Lambda = \text{diag}(\sigma_1^2, \ldots, \sigma_n^2)$이라 하면 $(A^TA)V = V\Lambda_1$이 성립하므로:

- **$V$의 열벡터들은 $A^TA$의 고유벡터** (우 특이벡터), $\sigma_i^2$이 그 고윳값
- **$U$의 열벡터들은 $AA^T$의 고유벡터** (좌 특이벡터)

```python
A_demo = np.array([[9.5, 1.4, 7, 0.3],
                   [4, 4, 9, 8],
                   [8, 9.1, 6.5, 9.3]])

U_d, S_d, Vt_d = np.linalg.svd(A_demo, full_matrices=True)

# V의 열벡터 = A^T A 고유벡터 검증
AtA_demo = A_demo.T @ A_demo
eigvals_d, eigvecs_d = np.linalg.eigh(AtA_demo)
idx_d = np.argsort(eigvals_d)[::-1]
eigvals_d = eigvals_d[idx_d]

print('일치 여부:', np.allclose(eigvals_d[:len(S_d)], S_d**2, atol=1e-6))  # True
```

## 3. SVD의 머신러닝에서의 응용: 영상압축

**rank-1 행렬**은 두 벡터의 외적으로 표현된다: $A = uv^T$. 예를 들어 $100 \times 100 = 10000$개 원소를 $2 \times 100 = 200$개로 표현할 수 있어 차원 축소에 활용된다.

**rank-k 근사**: $m \times n$ 행렬 $A$의 SVD가 $A = U\Lambda V^T$이고 $\text{rank}(A) = k$라 하면

$$A = \sigma_1u_1v_1^T + \sigma_2u_2v_2^T + \cdots + \sigma_ku_kv_k^T$$

특이값이 빠르게 감소하는 경우, 상위 몇 개($\sigma_1, \sigma_2$)만으로도 원본을 근사할 수 있다.

```python
def svd_approx(A, k):
    """상위 k개 특이값만 사용한 rank-k 근사"""
    U, S, Vt = np.linalg.svd(A, full_matrices=False)
    return sum(S[i] * np.outer(U[:, i], Vt[i, :]) for i in range(k))

A_comp = np.array([[3, 1, 4, 1], [5, 9, 2, 6], [5, 3, 5, 8], [9, 9, 9, 3]], dtype=float)
U_c, S_c, Vt_c = np.linalg.svd(A_comp)
print('σ₁ + σ₂가 전체의', round((S_c[0]+S_c[1])/S_c.sum()*100, 1), '% 차지')
```

4x4 행렬을 rank-1부터 rank-4까지 근사했을 때, 근사에 사용하는 특이값 개수가 늘어날수록 원본에 가까워지고 Frobenius 오차가 줄어든다.

![rank-k 근사: 특이값 개수에 따른 원본 복원도](/images/svd/03_SVD_압축_실습_행렬을_rank_k_근사로_복원.png)

**실제 영상 압축 예시**: 203×233 크기 흑백영상($\text{rank}=5$)을 5개 특이값만으로 압축하면, 원본 저장 공간 $203\times233=47299$가 $5\times(203+233)=2180$이 되어 **약 21.7배 압축**이 가능하다.

13×13 픽셀 하트 이미지로 직접 실습해보면, rank 수가 늘어날수록 압축 원소 수는 늘지만 화질이 원본에 가까워지는 트레이드오프를 확인할 수 있다.

```python
U_h, S_h, Vt_h = np.linalg.svd(heart, full_matrices=False)

for k in [1, 2, 3, 4, 5]:
    Hk = svd_approx(heart, k)
    compressed = k * (heart.shape[0] + heart.shape[1])
    # rank가 커질수록 compressed/원본 크기 비율도 커짐
```

![SVD 영상 압축: 특이값 수에 따른 화질 변화](/images/svd/04_실제_이미지_SVD_압축_시뮬레이션_하트_이미지_재현.png)

압축률과 화질(Frobenius 오차, 누적 에너지)의 관계를 그래프로 보면, 적은 수의 특이값만으로도 대부분의 정보(에너지)를 보존할 수 있다는 걸 확인할 수 있다.

## 핵심 요약

| 개념 | 정의 | 핵심 포인트 |
|---|---|---|
| PSDM | $x^TAx \geq 0$ | $A^TA$, $AA^T$는 항상 PSDM |
| SVD | $A = U\Lambda V^T$ | 모든 $m \times n$ 행렬에 적용 가능 |
| 특이값 | $\sigma_i = \sqrt{\lambda_i(A^TA)}$ | $\sigma_1 \geq \sigma_2 \geq \cdots \geq 0$ |
| V의 열벡터 | $A^TA$의 고유벡터 | 우 특이벡터 |
| U의 열벡터 | $AA^T$의 고유벡터 | 좌 특이벡터 |
| rank-k 근사 | $A_k = \sum_{i=1}^k \sigma_iu_iv_i^T$ | 상위 k개 특이값으로 근사 |
| 영상 압축 | $k(m+n) \ll mn$ | 특이값이 빠르게 감소할수록 효율적 |

### 머신러닝과의 연결

- **PCA (주성분 분석)**: 공분산 행렬의 SVD → 분산이 큰 방향으로 차원 축소
- **잠재 의미 분석 (LSA)**: 텍스트 행렬의 SVD → 문서-단어 관계 분석
- **추천 시스템**: 사용자-아이템 행렬의 SVD → 잠재 요인 모델
- **이미지 압축**: 픽셀 행렬의 SVD → 상위 특이값만으로 이미지 복원
- **노이즈 제거**: 작은 특이값은 노이즈에 해당 → 제거하면 신호 개선
